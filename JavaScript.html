📖 JavaScript란?

JavaScript는 웹 페이지에 동적인 기능을 추가할 수 있는 프로그래밍 언어입니다.

🏗️ JavaScript의 역할

  1. 사용자 입력에 따라 페이지 내용 변경
  2. 애니메이션 효과 추가
  3. 폼 검증 기능 제공
  4. 서버와 데이터 통신 (AJAX)
  5. 웹 애플리케이션 개발 가능

🧠 주요 기능

  1. 변수와 데이터 타입
  2. 조건문과 반복문
  3. 함수와 이벤트 처리
  4. DOM(Document Object Model) 조작
  5. 비동기 처리 (Promise, async/await)

🚀 정리

JavaScript는 HTML, CSS와 함께 웹 개발의 3대 요소 중 하나입니다.

동적인 웹 페이지와 다양한 기능을 구현할 수 있는 강력한 도구입니다.
