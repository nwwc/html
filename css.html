CSS란?

📖 CSS(Cascading Style Sheets)란?

CSS는 웹 페이지의 스타일과 레이아웃을 정의하는 스타일 시트 언어예요.

HTML이 웹 페이지의 뼈대를 만든다면, CSS는 그 뼈대에 색, 글꼴, 배치 등의 스타일을 입혀요.

🏗️ CSS의 역할

  1. 색상과 배경 (color, background 속성)
  2. 글꼴과 텍스트 스타일 (font-family, font-size, line-height)
  3. 레이아웃 설정 (display, position, flex, grid)
  4. 애니메이션과 전환 효과 (animation, transition)
  5. 반응형 디자인 (media queries)

🧠 주요 기능

  1. 선택자(Selector)를 사용해 특정 HTML 요소에 스타일을 적용할 수 있어요.
  2. 상속(Inheritance)으로 부모 요소의 스타일이 자식 요소에 전달돼요.
  3. 우선순위(Specificity)를 통해 충돌 시 적용할 스타일을 결정해요.
  4. 미디어 쿼리(Media Queries)를 사용해 반응형 웹을 구현할 수 있어요.

🚀 정리

CSS는 HTML에 스타일을 적용해 웹 페이지를 더 아름답고 사용하기 편하게 만들어줘요.
