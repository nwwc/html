HTML이란?

📖 HTML(HyperText Markup Language)이란?

HTML은 웹 페이지의 구조를 정의하는 마크업 언어예요.

🏗️ HTML의 역할

  1. 제목과 본문 (h1, p 태그)
  2. 이미지 (img 태그)
  3. 링크 (a 태그)
  4. 목록 (ul, ol, li 태그)
  5. 표 (table 태그)
  6. 입력 폼 (form, input, button 태그)

🧱 HTML의 주요 특징

  1. 구조화된 문서 생성
  2. 링크 기능 제공
  3. 미디어 통합 가능
  4. 폼과 입력 요소 제공

🚀 정리

HTML은 웹 페이지의 뼈대를 만들어주는 언어예요.

CSS와 JavaScript를 통해 더 아름답고 기능적인 웹을 만들 수 있어요!
